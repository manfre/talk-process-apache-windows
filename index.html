What's Wrong With Threads?

  • Uses more resources
  • Most pages have lots of IO
    • Network (Request/Response)
    • Cache
    • Database
    • Disk (Templates)
  • "Understanding the GIL" - David Beazley http://www.dabeaz.com/GIL/
    • Threads release their lock when blocking for IO
